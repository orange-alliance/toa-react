# The Orange Alliance

## Setup

1. Install the latest version of Node.js from [nodejs.org](https://nodejs.org/en/download/)
2. Run `npm install`

## Running locally

Run `npm run dev` and open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
